Table almighty 1.0 ( A react-table like library in AnjuarJS-1.0 )

  Library repo

Created this configurable table library in 2017 in AngularJS 1.0 for a organaization. After my organazation got merged made it as a open-source.

Features and usage
  • Sorting
  • Search
  • Selectable Search
  • Custom coloring
  • Bulk row action
  • Reordering
  • Pagination

Capable to sort colums which contains

  • alphabets
  • numbers
  • dates
